# Smart Obstacle Detection System using Arduino

## Description
This project detects objects using an ultrasonic sensor and displays them using a radar interface.

## Components
- Arduino
- Ultrasonic Sensor (HC-SR04)
- Servo Motor

## Working
The servo motor rotates the ultrasonic sensor, measures distance, and sends data to Processing for visualization.

## Software Used
- Arduino IDE
- Processing

## Applications
- Car parking assistance
- Robot navigation
- Safety systems
